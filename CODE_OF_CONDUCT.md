# Code of Conduct

This project should be a useful place to discuss research code, numerical
results, and implementation details.

## Expected behavior

- Be respectful and specific.
- Critique claims, methods, and code without attacking people.
- Assume good faith, but ask for evidence when claims are unclear.
- Keep discussions relevant to fluctuation-regularized optimization,
  reproducibility, and project maintenance.
- Accept that negative results and failed experiments are valuable.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Dismissive comments about a contributor's background or experience.
- Publishing private information without permission.
- Repeatedly derailing issues or pull requests.
- Misrepresenting benchmark results or hiding known limitations.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or
decline contributions that violate this code of conduct.

If you need to report a problem, use GitHub's reporting tools or contact the
maintainer through the repository owner profile.
