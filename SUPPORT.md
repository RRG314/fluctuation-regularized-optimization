# Support

Use GitHub issues for support.

## Good issue topics

- install problems;
- failing tests;
- unclear documentation;
- reproduction problems with benchmark scripts;
- questions about the implemented mechanisms;
- bug reports with a minimal example.

## Not support requests

- requests to prove universal optimizer superiority;
- unrelated optimizer ideas;
- requests for production deployment guarantees;
- broad research debates without a concrete repo action.

## Before opening an issue

Please include:

- operating system;
- Python version;
- install command;
- full error message or traceback;
- command used to reproduce the problem;
- whether the issue appears on a clean virtual environment.
